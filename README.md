# Interactive Landing Page
A responsive single-page landing built with HTML5 and CSS3. Includes navigation menu, hero section, body content, and footer. Implemented basic interactions. Fully responsive for desktop.
# Live Demo
(https://my-second-landing.netlify.app/)
# Run Locally
1. Clone the repo: git clone https://github.com/Z1g1e/frontend-portfolio.
2. Install dependencies: npm install.
3. Run development server: npm start.
